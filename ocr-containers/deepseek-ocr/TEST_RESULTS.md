## 座標変換の検証

### 入力 (DeepSeek形式)
```
<|det|>[[40, 55, 963, 135]]<|/det|>
```

### 変換プロセス
1. **正規化座標の抽出**: [40, 55, 963, 135]
2. **正規化解除** (画像サイズ 1000x1000):
   - x1 = 40/999 * 1000 = 40
   - y1 = 55/999 * 1000 = 55
   - x2 = 963/999 * 1000 = 963
   - y2 = 135/999 * 1000 = 135
3. **4点座標生成**:
   - 左上: [40, 55]
   - 右上: [963, 55]
   - 右下: [963, 135]
   - 左下: [40, 135]

### 出力 (PaddleOCR形式)
```json
"points": [[40, 55], [963, 55], [963, 135], [40, 135]]
```

✅ **変換成功**: 座標が正しく変換されている

## HTML除去の検証

### 入力
```html
<table><tr><td>氏名</td><td>日本</td><td>花子</td><td>昭和61年</td><td>5月</td><td>1日生</td></tr></table>
```

### 処理
1. `<br/>` → スペース
2. `</tr>` → スペース
3. `</td>` → スペース
4. その他のタグ → 除去
5. 連続スペース → 単一スペース

### 出力
```
氏名 日本 花子 昭和61年 5月 1日生
```

✅ **除去成功**: HTMLタグが完全に除去され、読みやすいテキストに変換されている
